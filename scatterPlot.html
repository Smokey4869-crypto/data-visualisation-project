About

The scatter plot represents the migration flow from states to states in the United States, covering the time period from 2010 to 2019. The plot includes an animation feature that automatically transitions the data to the next year, allowing the observation for the changes in migration flow. Users can hover over any data point to view detailed statistics in a tooltip, providing insights into specific state's migration patterns. Additionally, users can mark specific data points by clicking on them, enabling focused analysis and comparison.
